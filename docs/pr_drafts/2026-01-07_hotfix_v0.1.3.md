# Hotfix v0.1.3

## Summary
Fixes cross-site authentication issues on platforms like Render where `NODE_ENV` might not be strictly set or where reverse proxies strip protocol information.

## Changes
- **Backend Main**: Enabled `trust proxy` (required for correct protocol detection behind load balancers).
- **Auth Controller**: Implemented robust production detection. Authenticated cookies (`SameSite=None; Secure`) are now enabled if:
  - `NODE_ENV` is 'production', OR
  - `FRONTEND_URL` starts with `https`.
- **Versioning**: Bumped to `0.1.3`.

## Testing
- [x] Verified logic handles local (http) and production (https) URLs correctly.
