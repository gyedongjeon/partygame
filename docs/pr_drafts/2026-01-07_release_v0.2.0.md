# Release v0.2.0

## Summary
Release v0.2.0 introduces key gameplay improvements including player display names and a game duration timer.

## Changelog

### Features
- **Display Names**: Players now see display names instead of UUIDs in the voting list and game over screen.
- **Game Timer**: Added a countdown timer during gameplay when 'Time Limit' is enabled. The game automatically ends (Imposter wins) when time expires.
- **Vote Button Fix**: Fixed an issue where the vote button would not update to "Vote Cast" after clicking.

### Hotfixes Included
- v0.1.4: Fixed TypeScript boolean inference build error.
- v0.1.3: Robust production environment detection (trust proxy, improved logic).
- v0.1.2: Login and Cookie security fixes (CORS, SameSite=None).
