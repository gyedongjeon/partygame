# Hotfix v0.2.4: Allow Vercel Preview Domains

## Summary
The previous strict CORS policy blocked Vercel Preview Deployments (e.g., `https://partygame-git-main...`). This hotfix allows all `*.vercel.app` origins dynamically.

## Changes
- **`backend/src/main.ts`**: Updated CORS origin check to allow any origin ending in `.vercel.app`.
