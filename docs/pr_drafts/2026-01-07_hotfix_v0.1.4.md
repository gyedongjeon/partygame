# Hotfix v0.1.4

## Summary
Fixes a TypeScript build error introduced in v0.1.3 where `process.env.NODE_ENV` or URL checks resulted in types not strictly assignable to `boolean`.

## Changes
- **Auth Controller**: Explicitly cast the `isProduction` check to `boolean` to satisfy TypeScript strictness.
- **Versioning**: Bumped to `0.1.4`.

## Testing
- [x] `npm run build` passes locally.
